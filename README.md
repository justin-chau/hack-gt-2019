# hack-gt-2019
Remote Gimbal for HackGT 2019

# Instructions for starting:

* Run headset_imu.py and headset_vid_stream.py on the computer running the headset display.

* Run gimbal_imu.py and gimbal_vid_stream.py on the computer running the gimbal.

* If there is difficulty accessing the Teensy through serial, run the following command: 'sudo chmod a+rw /deve/ttyACM0'.
